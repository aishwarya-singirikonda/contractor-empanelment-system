# Contractor Empanelment System

A full-stack web application for managing contractor onboarding, departments, and document uploads.

## Features

- Admin Login Authentication (JWT)
- Contractor Management
- Department Management
- Document Upload Management
- Dashboard Analytics
- Status Tracking (Approved / Pending / Rejected)
- Pie Chart Visualization
- Protected Routes
- Responsive UI with Tailwind CSS

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Router
- Axios
- Chart.js

### Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL
- JWT Authentication

## Project Structure

contractor-empanelment-system/
├── frontend/
├── backend/
└── prisma/

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
